NotebookLM · Source Brief
Build the source notebook for [TOPIC]
Create a NotebookLM notebook titled "[TOPIC] — Momentum Source Pack". Upload these source types only: 1. Peer-reviewed studies (PubMed, full-text PDFs preferred) 2. Society guidelines (e.g., AACE, Endocrine Society, IFM) 3. Reputable functional-medicine textbooks or whitepapers 4. Existing Momentum blog posts on adjacent topics Do NOT upload: - Patient charts, names, or any PHI - Reddit, social posts, or anonymous web forums - Sources older than 10 years unless landmark Once uploaded, ask NotebookLM to: - Summarize the consensus view in 5 bullets - Flag any contradictions between sources - Generate an audio overview targeted at a curious patient (not a clinician) - Draft 3 candidate hooks for an Instagram carousel grounded in these sources, each citing the source #
NotebookLM · Infographic
Generate a single-image infographic for [TOPIC]
Using only the sources in this notebook, generate one shareable infographic for Instagram and Facebook. Format requirements: - Aspect ratio: 1:1 (square) primary; also produce a 4:5 (portrait) version - Title: short, benefit-led, no clickbait - 3 to 5 stat or insight blocks, each citing the source number from the notebook - Footer: "Source citations available — DM us" plus the Momentum logo placeholder - Color palette: deep charcoal background (#1c1c1c), gold accent (#C9A84C), warm white text - Tone: clinical-but-warm, never alarmist - Avoid: prescriptive medical advice, before/after claims, weight-loss promises, HIPAA-sensitive imagery Output the infographic, then list the exact source citations used so we can verify before publishing.
Claude · IG Carousel
Educational Instagram carousel about [TOPIC]
You are writing for Momentum Health & Wellness MN, a functional medicine and chiropractic clinic in Elk River, MN. Voice: clinical-but-warm, root-cause minded, never alarmist, never prescriptive. Write a 6-slide Instagram carousel about [TOPIC]. Per-slide format: - Slide 1: Hook (max 8 words). One subline. - Slides 2–5: One insight each. Headline (max 6 words) + 2 short sentences. Cite a source if you make a claim. - Slide 6: CTA — "Free 15-min phone consult — link in bio." Add one warm closing sentence. Then below the slides, output: - A 120–180 word IG caption (no emojis at start; one or two mid-caption is fine) - 12 hashtags relevant to functional medicine + the topic, mixed reach - Suggested alt text for each slide Hard rules: - No PHI, no patient names, no before/after claims - No "cures" — we address root causes - Cite sources peer-reviewed or society-level only - Avoid weight-loss-as-aesthetic framing
Claude · FB Long-Form
Patient-education Facebook post about [TOPIC]
Write a 350–500 word Facebook post for Momentum Health & Wellness MN. Topic: [TOPIC] Structure: 1. Open with a relatable scenario (no patient identifiers, no PHI). Generic "many people we see…" framing. 2. Explain the conventional view in one paragraph. 3. Explain the functional-medicine root-cause view in one paragraph. 4. Three-bullet "what we actually look at" list (labs, history, lifestyle drivers). 5. One paragraph on what change can look like, kept honest — no guarantees. 6. Close with a soft CTA: free 15-minute phone consult, link in comments. Tone: clinical-but-warm. Cite 1–2 peer-reviewed sources inline by author + year. No alarm, no scare tactics, no "cure" language. Then list: - A 1-line preview text (under 90 chars) for link-share - Suggested image direction (one sentence) so the designer knows what hero shot to make
Claude · Reel Script
30-second Reel hook + script for [TOPIC]
Write a 30-second Instagram Reel for Momentum Health & Wellness MN. Topic: [TOPIC] Output exactly this structure: - HOOK (0–3 sec): one line, max 7 words, pattern-interrupt - BODY (3–25 sec): 3 beats. Each beat is one short on-camera sentence + one b-roll cue in brackets, e.g. [b-roll: Dr. pointing at thyroid model] - PAYOFF (25–30 sec): one-line reframe + soft CTA - ON-SCREEN TEXT: 4–6 short overlays timed to each beat - CAPTION: 80–120 words, ends with "Free 15-min phone consult — link in bio" - 8 hashtags Voice: warm, direct, no jargon, no alarm. Patient-first. No PHI, no diagnosis-by-Reel, no "cure" claims.
Nano Banana · Imagery
Brand-consistent hero image for [TOPIC]
Generate a hero image for an Instagram + Facebook post on [TOPIC] for a functional-medicine and chiropractic clinic in Minnesota. Composition: - Subject: [SUBJECT — e.g. soft-focus close-up of healthy whole foods, clinical hands gently examining, calm Midwest natural light] - Aspect ratio: 1:1 square (primary) and 9:16 portrait (Reel cover variant) - Style: editorial wellness photography, shallow depth of field, warm but clinical - Palette: deep charcoal (#1c1c1c) and warm gold (#C9A84C) accents, never neon, never overly saturated - Mood: calm, trustworthy, honest, never staged-stocky - Lighting: golden-hour or soft window light Avoid: - Generic stock-photo poses (white-coats-with-stethoscopes-pointing-at-tablets, etc.) - Before/after framings, weight-loss tropes - Watermarks, text overlays (text gets added later) - Faces in detail unless we approved a model Output 4 variants so we can pick the strongest.

Brand Kit

Stay On-Brand

The same tokens used across the public Momentum site. Reuse them in every prompt and template.

Color Palette

Gold
G-dark
G-light
Black
Charcoal
White
  • Gold: #C9A84C — primary accent
  • Gold dark: #A07830 — hover / depth
  • Charcoal: #1c1c1c — backgrounds
  • Black: #0d0d0d — heroes

Typography

  • Playfair Display — headlines, doctor names, hero copy
  • Inter — body, captions, UI
  • Italic Playfair for accent words (matches public site)
  • All-caps Inter (letter-spacing 0.18em) for labels

Voice — Do / Don't

  • Root-cause, clinical-but-warm
  • Cite peer-reviewed sources
  • "Many people we see…"
  • "Cures," "miracles," guaranteed outcomes
  • Patient names, photos without consent, any PHI
  • Before/after weight-loss framing
